# Inputs

이 환경은 한 명의 지원자 프로필을 지속적으로 재사용한다.

- `private/sources/`: 사용자가 제공한 원본 문서. 에이전트는 수정하지 않는다.
- `profile/`: 승인된 기준 프로필, 근거 장부, 자산과 변경 이력.
- `examples/`: 공유 가능한 익명 드라이런 자료.

최초 사용 시 개인 자료를 첨부하고 자연어로 프로필 생성을 요청한다. 에이전트가 `profile/*.template.md`를 사용하며 사용자가 템플릿을 직접 편집할 필요는 없다. 실제 프로필 파일과 개인 원본은 `.gitignore`로 공유 대상에서 제외된다.
