# Job Papers Optimize

한 명의 지원자가 여러 회사와 직무에 지원할 때 사용하는 프로젝트 로컬 AI 작업환경이다. 특정 에이전트 제품, 전역 스킬, 개발 도구 또는 이전 작업환경에 의존하지 않는다.

## 시작하기

이 환경은 Claude Code, Codex, Antigravity처럼 로컬 폴더를 읽고 쓸 수 있는 AI 에이전트에서 사용한다. 별도 프로그램 설치, 터미널 명령 또는 주문서 작성은 필요하지 않다.

1. 내려받은 압축을 풀고 `job_papers_optimize` 폴더를 찾는다.
2. Claude Code, Codex 또는 Antigravity에서 **폴더 열기**, **프로젝트 열기**, **워크스페이스 열기**에 해당하는 메뉴를 선택한다.
3. `AGENTS.md`, `inputs`, `orders`, `outputs`, `skills`가 바로 보이는 `job_papers_optimize` 폴더를 연다.
4. 해당 폴더에서 새 대화나 작업을 시작하고 아래처럼 말한다.

> 이 폴더의 AGENTS.md와 로컬 skills를 먼저 읽고 따라줘. 처음 사용하는 환경이니 내 지원자 프로필부터 만들어줘.

개인 자료는 대화에 직접 첨부하거나 `inputs/private/sources/`에 넣는다. AI 환경에 파일 첨부 기능이 없다면 폴더에 넣는 방식이 가장 확실하다. 원본 파일은 에이전트가 수정하지 않는다.

채용공고는 파일, URL 또는 복사한 본문 중 편한 방식으로 제공하면 된다. 사용자가 주문서나 YAML 파일을 직접 만들 필요는 없다.

## 사용 예시

최초 사용 예시:

> 내 이력서와 포트폴리오를 첨부했어. 앞으로 여러 회사 지원에 재사용할 지원자 프로필을 만들어줘.

회사 지원 예시:

> 이번에는 아크랩 Product Designer에 지원하려고 해. 채용공고를 첨부했어. 기존 프로필을 사용해서 이력서와 포트폴리오를 Markdown으로 최적화하고 면접도 준비해줘.

프로필 갱신 예시:

> 새 프로젝트 자료를 추가했어. 기존 프로필에서 달라질 내용만 제안해줘. 아직 기준 프로필에는 반영하지 마.

에이전트는 필요한 경우에만 회사, 직무, 채용공고, 지원 단계, 원하는 결과물을 짧게 확인하고 내부 주문을 자동으로 만든다.

## 기본 동작

- 입력: 현재 AI 환경이 읽을 수 있는 PDF, DOCX, Markdown, HTML, JSON, TXT
- 기본 출력: Markdown
- 선택 출력: 런타임이 지원하는 경우 DOCX 또는 HTML
- PDF: 입력 전용
- 원본과 개인 자료: 읽기 전용
- 프로필 갱신: 변경안 검토 후 명시적 승인

현재 AI 환경이 첨부 문서 형식을 읽지 못하면 설치를 요구하지 않고 PDF, Markdown, TXT 등 적절한 대체 형식을 요청한다. DOCX 출력 기능이 없으면 같은 승인 내용을 Markdown으로 제공하고 그 제한을 결과에 기록한다.

## 반복 사용

최초 프로필을 승인한 뒤에는 회사별 지원 때 원본 자료를 다시 분석하지 않는다. 회사와 직무가 바뀌면 승인된 프로필에서 적합한 경험을 선택하고 강조 순서와 표현만 조정한다. 새 경력이나 프로젝트가 생겼을 때만 별도 갱신안을 만든다.

## 개인정보

`inputs/private/`, 실제 `inputs/profile/` 파일, `outputs/`는 공유하거나 버전 관리하지 않는다. 배포본에는 템플릿과 익명 예시만 포함한다.

## 이용 범위와 라이선스

이 프로젝트는 개인 연구, 학습, 실험과 자신의 채용 지원 준비를 위해 사용할 수 있다. 자신의 이력서·포트폴리오·자기소개서·면접 자료를 만들고 이를 취업에 제출하거나, 그 결과로 급여를 받는 것은 허용되는 개인 사용이다.

다음과 같은 영리 이용은 금지한다.

- 프로젝트 자체나 수정본을 판매, 재판매, 임대 또는 유료 배포하는 행위
- 이 프로젝트를 이용해 타인에게 유료 이력서 작성, 취업 컨설팅, 코칭, 채용 대행 또는 교육을 제공하는 행위
- 프로젝트를 유료 제품, SaaS, 플랫폼, 사내 상업 서비스 또는 수익화된 워크플로우에 포함하는 행위
- 프로젝트 접근권, 실행 결과 또는 이를 기반으로 한 서비스를 대가를 받고 제공하는 행위

사용자가 제공한 이력서, 포트폴리오, 경력 정보와 개인정보에 프로젝트가 권리를 주장하지 않는다. 개인 맞춤 산출물은 자신의 구직 목적으로 사용할 수 있다.
